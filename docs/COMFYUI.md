# 🎨 ComfyUI Integration

> ComfyUI is the primary image generation engine used by AAA Infinity AI.

## Purpose

ComfyUI provides a powerful node-based workflow system for AI image generation.

Ari AI connects to ComfyUI running on Google Colab.

---

## Supported Models

- Stable Diffusion 1.5
- SDXL
- FLUX (Future)
- Realistic Vision
- Juggernaut XL
- DreamShaper
- Anything V5
- Absolute Reality

# 🚀 Hugging Face Spaces

> Connect and use Hugging Face Spaces inside AAA Infinity AI.

## Overview

Hugging Face Spaces allow Ari AI to use community-built AI applications.

Supported frameworks

- Gradio
- Streamlit
- Docker

---

## Supported Tasks

- Text Generation
- Image Generation
- Video Generation
- Image Editing
- Background Removal
- OCR
- Speech Recognition
- Text To Speech
- Music Generation

---

## Connector

The Hugging Face Spaces Connector can

- Save Space IDs
- Connect to Gradio APIs
- Detect API Endpoints
- Check Space Status
- Retry Failed Requests
- Cache Results

---

## Configuration

Store securely using Infisical.

HF_TOKEN

HF_SPACE_ID

HF_SPACE_URL

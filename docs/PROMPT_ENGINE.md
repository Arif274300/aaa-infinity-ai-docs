# ✨ Prompt Engine

> The intelligent prompt processing system of AAA Infinity AI.

## Purpose

The Prompt Engine transforms a simple user request into a high-quality prompt optimized for the selected AI provider.

Ari AI uses the Prompt Engine before sending requests to OpenRouter, Hugging Face, Google Colab or other providers.

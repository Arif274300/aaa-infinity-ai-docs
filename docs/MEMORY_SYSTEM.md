# 🧠 Memory System

> The long-term memory architecture of Ari AI.

## Purpose

The Memory System enables Ari AI to provide a more personalized experience while keeping users in control of what is stored.

---

# Memory Types

## Session Memory

Stores information only during the current conversation.

Automatically cleared when the session ends.

---

## Project Memory

Stores

- Chats
- Images
- Videos
- Files
- Prompts
- Workflows

inside a project.

---

## User Preferences

Stores

- Theme
- Language
- Favorite Models
- Favorite Providers
- Preferred Image Style
- Preferred Video Style

---

## Prompt History

Stores

- Previous Prompts
- Prompt Versions
- Favorite Prompts
- Prompt Templates

---

## Gallery History

Stores

- Images
- Videos
- Audio
- Documents

generated by Ari AI.

---
